Detailed Description

Ventana principal de la aplicación de gestión de conciertos.

Esta clase representa la interfaz gráfica del menú principal, que permite al usuario acceder a las funcionalidades de registro, inicio de sesión y visualización del catálogo de eventos. Utiliza componentes de Swing para crear una interfaz amigable con botones estilizados y un diseño centrado.
