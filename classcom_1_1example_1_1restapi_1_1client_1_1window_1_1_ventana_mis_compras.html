Detailed Description

Ventana gráfica que muestra las compras realizadas por un usuario autenticado.

Esta clase construye una interfaz con Swing que se conecta a una API REST para obtener las compras realizadas por un usuario y las muestra en una tabla. Realiza múltiples solicitudes GET para recuperar tanto los datos de las compras como los detalles de los conciertos.
