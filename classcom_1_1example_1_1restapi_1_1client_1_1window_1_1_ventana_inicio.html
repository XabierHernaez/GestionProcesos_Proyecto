Detailed Description

Ventana de inicio de sesión para los usuarios del sistema.

Permite a los usuarios ingresar su correo electrónico y contraseña para autenticarse a través de una API REST. Si la autenticación es exitosa, redirige a la ventana correspondiente según el tipo de usuario (ADMIN o CLIENTE).
